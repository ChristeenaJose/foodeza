Speicher: Bestellungen aus market.html liegen im Browser unter foodeza_online_orders (früher foodeza_market_orders — wird beim ersten Aufruf übernommen). Zusätzlich werden beim Laden Bestellungen aus der Datei files/generated/online-order/market/orders.json eingespielt (falls auf dem Server vorhanden). Hinweis: Ohne eigenes Backend erscheinen Handy-Bestellungen nur auf dem jeweiligen Gerät — für eine zentrale Datei auf dem Server die JSON dort per FTP/deploy aktualisieren oder JSON importieren.
